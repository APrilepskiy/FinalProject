Тестируемый сайт https://lms.skillfactory.ru/
Для запуска тестов используется класс MyTest.java

Тестами проверяется: 
функционал входа в личный кабинет, 
функционал регистрации, 
просмотр курсов, 
поиск курсов, 
работоспособность кнопок в футоре,
работоспособность кнопок навигации по сайту,
проверяется корректное открытие страниц при переходе из главной страницы.

Для каждой страницы создан свой класс, где описанны элементы для взаимодействия со страницей и методы для текущей страницы.
Для хранения тестовых данных создан класс TestValues.java

ВНИМАНМИЕ!!!
Для тестирования в переменные loginForTest, passwordForTest, userName необходимо ввести актуальные данные в классе TestValues.java
Можно и не вводить, но без них тест просто упадёт)
